# Skinhub
A skincare e-commerce website
how to run this project
Download This Project Folder and Extract it Move to project folder in Terminal. Then run following Commands : python manage.py makemigrations python manage.py migrate python manage.py runserver

Now enter following URL in Your Browser Installed On Your Pc http://127.0.0.1:8000/
